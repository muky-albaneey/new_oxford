# Hero Video Production Checklist for Oxford Manor College

## Pre-Production

### Planning
- [ ] Review video script and concept
- [ ] Identify key locations on campus
- [ ] Schedule filming dates (avoid exam periods)
- [ ] Obtain necessary permissions (students, parents, staff)
- [ ] Prepare equipment list
- [ ] Create shot list with timings
- [ ] Coordinate with school events/activities

### Content Planning
- [ ] Identify 5-10 key shots to capture
- [ ] Plan for diverse student representation
- [ ] Include various activities (academic, sports, arts)
- [ ] Capture campus beauty shots
- [ ] Plan for smooth transitions between shots

## Production Day

### Equipment Checklist
- [ ] Camera (DSLR/Mirrorless or professional)
- [ ] Multiple lenses (wide, standard, telephoto)
- [ ] Stabilization (gimbal/tripod)
- [ ] Extra batteries and memory cards
- [ ] Lighting equipment (if needed)
- [ ] Drone (if permitted and licensed)

### Shot List
- [ ] Wide campus aerial/establishing shot
- [ ] Main building entrance
- [ ] Science laboratory (students working)
- [ ] Library (students studying)
- [ ] Classroom (engaged learning)
- [ ] Sports field (active sports)
- [ ] Art room (creative activities)
- [ ] Assembly hall or theatre
- [ ] Student social areas
- [ ] Closing campus shot

### During Filming
- [ ] Ensure good lighting (natural preferred)
- [ ] Capture smooth, steady shots
- [ ] Get multiple angles of key scenes
- [ ] Film extra B-roll for flexibility
- [ ] Maintain consistent color temperature
- [ ] Ensure all shots are in focus

## Post-Production

### Editing
- [ ] Import and organize footage
- [ ] Select best takes for each shot
- [ ] Create smooth transitions
- [ ] Color grade to match brand (#728639, #d4af37)
- [ ] Add subtle text overlays (optional)
- [ ] Ensure seamless loop point
- [ ] Optimize for web delivery

### Export Settings
- [ ] Format: MP4 (H.264)
- [ ] Resolution: 1920x1080 or higher
- [ ] Frame rate: 24fps or 30fps
- [ ] Bitrate: Optimized for web (2-5 Mbps)
- [ ] File size: Under 10MB for 30 seconds
- [ ] Test playback on multiple devices

### Quality Check
- [ ] Video loops seamlessly
- [ ] No audio (muted)
- [ ] Colors match brand
- [ ] Smooth playback on all devices
- [ ] Fast loading time
- [ ] Fallback image works

## Deployment

### File Placement
- [ ] Upload to `/public/videos/hero-school.mp4`
- [ ] Create WebM version (optional)
- [ ] Test on staging environment
- [ ] Verify autoplay works
- [ ] Check mobile performance
- [ ] Monitor page load speed

### Testing
- [ ] Desktop browsers (Chrome, Firefox, Safari, Edge)
- [ ] Mobile devices (iOS, Android)
- [ ] Tablet devices
- [ ] Different connection speeds
- [ ] Video fallback behavior

## Alternative Solutions

If professional video isn't available:

### Option 1: Stock Video
- [ ] Search Pexels/Pixabay for school/education videos
- [ ] Select 3-5 high-quality clips
- [ ] Edit together with smooth transitions
- [ ] Color grade to match brand

### Option 2: Image Slideshow
- [ ] Use 5-10 high-quality campus photos
- [ ] Create smooth crossfade transitions
- [ ] Add subtle zoom/pan effects
- [ ] Loop seamlessly

### Option 3: Animated Background
- [ ] Use CSS animations
- [ ] Create parallax effects
- [ ] Add subtle movement to static images
- [ ] Maintain premium feel

## Budget Considerations

### Professional Production
- Videographer: $500-$2,000/day
- Equipment rental: $200-$500/day
- Post-production: $1,000-$3,000
- **Total: $1,700-$5,500**

### DIY Production
- Equipment (if purchasing): $1,000-$3,000
- Editing software: $0-$300
- Time investment: 20-40 hours
- **Total: $1,000-$3,300 (one-time)**

### Stock Video
- Free options available
- Premium stock: $50-$200
- Editing time: 5-10 hours
- **Total: $0-$200**
